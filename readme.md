# Desafio Técnico - Integração Supabase + Z-API

Este projeto foi desenvolvido como parte do processo seletivo para a vaga de Estágio em Desenvolvimento Python na b2bflow. A aplicação realiza a leitura de contatos integrados em um banco de dados Supabase e realiza o disparo automatizado de mensagens personalizadas via WhatsApp utilizando a Z-API.

## 🚀 Tecnologias Utilizadas

* **Python 3**
* **Supabase** (Banco de dados PostgreSQL na nuvem)
* **Z-API** (Gateway de integração com o WhatsApp)
* **python-dotenv** (Gerenciamento seguro de variáveis de ambiente)
* **requests** (Consumo de APIs REST)
